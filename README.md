# Inti Suite

Sistema integrado de gestão para engenharia, energia solar, obras, projetos, clientes e automações.

## Objetivo

Centralizar em uma única plataforma:

* Agenda
* CRM
* Propostas
* Contratos
* Obras
* Relatórios
* Pós-venda
* Energia Solar
* Inteligência Artificial

## Tecnologias

Frontend:

* React
* Vite
* TypeScript
* Tailwind CSS

Backend:

* FastAPI
* Python

Banco de Dados:

* PostgreSQL (Supabase)

Autenticação:

* Supabase Auth
